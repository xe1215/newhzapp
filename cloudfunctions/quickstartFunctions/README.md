# QuickStart Reference

This folder is kept only as a development reference for the original WeChat CloudBase quickstart.

The mini program no longer registers QuickStart pages as user-facing entry points. New product code should call the business cloud functions under `cloudfunctions/user`, `cloudfunctions/test`, `cloudfunctions/report`, `cloudfunctions/payment`, `cloudfunctions/share`, and `cloudfunctions/cleanupExpiredData`.
