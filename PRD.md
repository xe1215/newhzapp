# 口红试色小程序 PRD

## 1. 产品概述

这是一个微信小程序口红试色与推荐工具。用户上传自拍并选择偏好后，系统生成 3 张免费试色预览图；用户不满意可免费换一组，最多 3 次；满意后支付 5.99 元，解锁当前预览组对应的完整口红推荐报告。

第一版只验证“自拍试色预览 -> 低价付费报告”的闭环，不做商城、社区、会员、优惠券和购买链接。

## 2. 目标用户

- 不确定自己适合什么口红色号的用户。
- 看过很多试色图，但无法判断自己上脸效果的用户。
- 需要通勤、约会、素颜、显白、气场等场景化建议的用户。
- 愿意为低价个性化美妆建议付费的用户。
- 运营人员：维护口红库、查看订单、处理异常退款、查看转化数据。

## 3. 核心流程

1. 用户进入首页，点击开始测试。
2. 上传自拍，系统做内容审核和图片质量检查。
3. 用户选择肤色、预算、场景、风格。
4. 系统按规则推荐 3 个口红，并生成一组 3 张水印试色预览图。
5. 用户查看免费预览；不满意可免费换一组，最多 3 次。
6. 用户支付 5.99 元。
7. 系统解锁当前激活预览组的完整报告。
8. 用户查看报告、保存结果卡片、分享单张卡片、回看我的报告。

## 4. 用户功能

| 功能 | 第一版要求 |
| --- | --- |
| 静默登录 | 使用微信 openid，不强制获取手机号、头像、昵称 |
| 自拍上传 | 支持拍照/相册；提示正脸、清晰、自然光、嘴唇无遮挡 |
| 图片检查 | 拦截违规、模糊、遮挡、非人脸、嘴唇不可识别等图片 |
| 偏好选择 | 肤色、预算、场景、风格 |
| 试色预览 | 每组 3 张水印图；只改嘴唇颜色；不展示品牌、色号、价格和推荐理由 |
| 免费换组 | 最多免费换 3 次；失败不消耗次数 |
| 支付解锁 | 5.99 元解锁当前激活预览组 |
| 完整报告 | 展示 3 个推荐、无水印试色图、品牌色号、价格区间、推荐理由、避雷点、平替、搜索关键词 |
| 我的报告 | 只展示已支付报告 |
| 保存卡片 | 将单个推荐生成结果卡片并保存到相册 |
| 分享 | 只分享单张结果卡片，不暴露完整报告 |
| 删除 | 用户可删除自拍原图；可隐藏自己的报告 |
| 退款 | 生成失败或支付后报告不可查看可退款；已成功查看完整报告不退款 |

| 功能 | 第一版要求 |
| --- | --- |
| 数据概览 | 访问量、测试数、订单数、收入 |
| 转化漏斗 | 上传、生成、换组、预览、支付点击、支付成功 |
| 分享数据 | 分享次数、访问人数、带来的测试数和支付数 |

| 口红库管理 | 维护 50-70 个精品色号及推荐字段 |
| 异常退款 | 查看失败原因并处理退款 |
| 操作日志 | 记录登录、修改、上下架、退款、查看图片等敏感操作 |

## 6. 预览换组规则

- 初始生成 1 组预览。
- 用户最多免费换 3 次，所以一次测试最多出现 4 组预览。
- 每组固定 3 张试色图。
- 换组成功才消耗次数；换组失败不消耗次数。
- 只有当前激活组可支付解锁。
- 旧预览组不可支付，也不可在支付后解锁。
- 达到上限后，提示用户修改偏好重新测试。

页面文案：

- 按钮：不太喜欢，换一组
- 次数：剩余 3/2/1 次免费换色机会
- 上限：本次免费换色机会已用完，你可以修改偏好重新测试

## 7. 完整报告内容

完整报告展示 3 个推荐角色：

- 最适合你
- 日常不出错
- 风格加分款

每个推荐包含：

- 无水印试色图
- 品牌色号
- 价格区间
- 推荐理由
- 避雷点
- 平替，可为空
- 通用搜索关键词
- “试色仅供参考”提示

## 8. 界面设计

整体风格：简洁、清透、美妆感，一眼能看懂当前步骤和下一步操作。

视觉要求：

- 白底为主，玫瑰红作为主按钮和重点色。
- 页面少装饰，重点突出自拍图、试色结果和支付按钮。
- 预览页使用 3 张结果卡片，不做冗长报告长图。
- 被锁住的报告内容用模糊、锁图标、标题占位表达，不展示付费信息。
- 分享页只展示单张结果卡片和重新测试入口。

页面：

- 首页
- 上传自拍页
- 偏好选择页
- 生成中页
- 免费预览页
- 支付结果页
- 完整报告页
- 我的报告页
- 分享落地页
- 隐私说明页
- 退款/客服说明页

## 10. 边界条件

- 不承诺 100% 真实还原色号。
- 试色图必须像用户本人，不得生成随机 AI 模特。
- 图像模型只用于生成试色图，不决定推荐结果。
- 推荐结果由口红库、用户偏好和人工权重决定。
- 免费预览不展示品牌、色号、价格、推荐理由、避雷点、平替、关键词。
- 分享只展示单张结果卡片，不展示完整报告。
- 自拍原图 24 小时后自动删除。
- 未支付草稿 24 小时后自动清理。
- 删除自拍不删除已生成报告。
- 隐藏报告不删除后台订单。
- 支付成功且报告可查看后不支持退款。

## 11. 验收标准

- 用户可完成上传、偏好、生成、换组、预览、支付、报告闭环。
- 每次测试最多免费换 3 次，次数展示正确。
- 换组失败不消耗次数。
- 支付只能解锁当前激活预览组。
- 支付成功后可立即查看完整报告。
- 试色图保留用户身份特征，且 3 个颜色有明显差异。
- 生成失败可重试，已支付但交付失败可退款。
- 后台可查看订单、测试记录、漏斗、分享数据和退款状态。
- 隐私说明、免责声明、权限规则完整。
